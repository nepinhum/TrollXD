## Installation & Basic Usage‚Äî‚ùî
* You can download the plugin as .phar from [releases](https://github.com/nepinhum/TrollXD/releases) and install it directly. You can also download it as a folder with the [download](https://github.com/nepinhum/TrollXD/archive/refs/heads/main.zip) option. Either way, all you have to do is put it in "root/plugins"
* If you want to add your language: (both options)
    * Open the "‚Ä¶/plugins/TrollXD/resources/locale-data" folder and create the language file you want to add (e.g. "tr_TR.json") and then you can add your own language based on an existing one!
      You can also open a ["pull request"](https://github.com/nepinhum/TrollXD/pulls) if you want to contribute and share your language with other users! <br>
      Also remember to change the "defaultLanguage" parameter in "config.yml" to the filename of the new language.
